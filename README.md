## Running the tests

> You can run the unit tests for this library from your command line using

```sh
npm nx test
```

## Code style

> we use the Airbnb JavaScript style guide, and Prettier as our linting tool.

**_to lint your code_**

```sh
npm nx lint
```

**_to automatically fix violations_**

```sh
npm nx lint:fix
```
